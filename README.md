# Community Intelligence Engine

A Gemini-powered reasoning system that transforms fragmented, multilingual community messages into actionable intelligence.

## Overview
Community Intelligence Engine uses Gemini as a core reasoning engine to infer emerging risks, priority needs, coordination signals, and reasoning explanations from noisy community data.
